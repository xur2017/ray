Time Series Forecasting¶

Time series forecasting is an interesting topic in Machine Learning. It requires attention and can also be beneficial if allied to other complex topics such as stock price prediction, weather forecasting, earthquake prediction and so on. Time series forecasting is the application of a model to predict future values based on previously observed values.

Time series problem involves a time component indexing data points in time order. Exploring the data/time relationship is key to the data analysis. Among the several approaches developed over the years, Recurrent Neural Network has shown significant application on the field. Here we are going to build RNN models with Pytorch to predict Amazon stock market price.

def split_data(data_raw, seq_len, train_pct):
    data = []
    
    # create all possible sequences of length seq_len
    for index in range(len(data_raw) - seq_len): 
        data.append(data_raw[index:index + seq_len])
    
    data = np.array(data);
    train_set_size = int(np.round(train_pct*data.shape[0]));
    
    x_train = data[:train_set_size,:-1]
    y_train = data[:train_set_size,-1]
    
    x_test = data[train_set_size:,:-1]
    y_test = data[train_set_size:,-1]
    
    return [x_train, y_train, x_test, y_test]

class GRU(nn.Module):
    def __init__(self, input_dim, hidden_dim, num_layers, output_dim):
        super(GRU, self).__init__()
        self.hidden_dim = hidden_dim
        self.num_layers = num_layers
        
        self.gru = nn.GRU(input_dim, hidden_dim, num_layers, batch_first=True)
        self.fc = nn.Linear(hidden_dim, output_dim)

    def forward(self, x):
        h0 = torch.zeros(self.num_layers, x.size(0), self.hidden_dim).requires_grad_()
        out, (hn) = self.gru(x, (h0.detach()))
        out = self.fc(out[:, -1, :]) 
        return out
