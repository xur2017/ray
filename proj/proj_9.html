House Price Estimation¶

Pricing is a common task among different sectors or services. For example, a real estate agent want help to objectively estimate house prices for different areas in US. He would like to give us one housing dataset to work on, and to see if machine learning technique is helpful to solve such problem. We are going to firstly check what useful information contained in the dataset, then build and train a machine learning model for estimating house prices for different areas.

The dataset contains the following columns:

  • 'Avg. Area Income' - average income of the householders in the area.
  • 'Avg. Area House Age' - average age of the houses in the area.
  • 'Avg. Area Number of Rooms' - average number of rooms for the houses in the area.
  • 'Avg. Area Number of Bedrooms' - average number of bedrooms for the houses in the area.
  • 'Area Population' - population of the area.
  • 'Price' - price of the house.
  • 'Address' - address of the house.

Set up the Training and Testing Datasets¶

In [6]:
from sklearn.model_selection import train_test_split
X = df[['Avg. Area Income', 'Avg. Area House Age', 'Avg. Area Number of Rooms',
        'Avg. Area Number of Bedrooms', 'Area Population']]
y = df['Price']
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2, random_state=1)

Mean Absolute Error (MAE): $\frac{1}{n}\sum_{i=1}^{n} |y_i-\hat{y_i}|$

Mean Squared Error (MSE): $\frac{1}{n}\sum_{i=1}^{n} (y_i-\hat{y_i})^2$

Root Mean Squared Error (RMSE): $\sqrt{\frac{1}{n}\sum_{i=1}^{n} (y_i-\hat{y_i})^2}$
